Como usar

Você pode usar o SASS tanto na Syntax SCSS quanto SASS. O que vai definir isso, é a extensão dos arquivos na pasta Sass.

Se salvar seu arquivo com a extensão .scss, você pode escrever o sass da maneira que vem usando. Se usar a extensão .sass, você deve seguir as regras de identação e a syntax da linguagem.

Tarefas Disponíveis

O projeto virá por padrão com algumas tarefas definidas no gulpfile.js.

Para ver as opções de tarefas disponíveis, você pode usar o comando gulp help.
